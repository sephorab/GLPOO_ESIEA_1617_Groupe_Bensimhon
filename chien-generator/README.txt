
Ceci est un projet Java-Maven à utiliser pour les TP de Génie Logiciel.

Ce petit programme permet de générer des fichiers CSV avec plein de chiens, dont les caractéristiques sont choisies aléatoirement. Le nombre de chiens (i.e. de lignes) dans le fichier est paramétrable. Je vous conseille de générer plusieurs fichiers en faisant varier le nombre de chiens. Cela vous fera autant de jeux de test.
